# 포트 충돌 해결 가이드

## 문제
`ERROR: [Errno 48] Address already in use` - 포트 8000이 이미 사용 중입니다.

## 해결 방법

### 방법 1: 기존 프로세스 종료 (추천)

```bash
# 포트 8000을 사용하는 프로세스 찾기
lsof -ti:8000

# 프로세스 종료
kill $(lsof -ti:8000)

# 또는 특정 PID로 종료
kill 60645
```

### 방법 2: 강제 종료

```bash
# 강제 종료
kill -9 $(lsof -ti:8000)
```

### 방법 3: 다른 포트 사용

`run.py` 파일을 수정하여 다른 포트 사용:

```python
uvicorn.run(
    "src.api.main:app",
    host="0.0.0.0",
    port=8001,  # 8000 대신 8001 사용
    reload=True,
    log_level="info"
)
```

그러면 `http://localhost:8001`로 접속하세요.

### 방법 4: 사용 가능한 포트 찾기

```bash
# 사용 가능한 포트 찾기
python -c "import socket; s = socket.socket(); s.bind(('', 0)); print(s.getsockname()[1])"
```

## 확인

포트가 해제되었는지 확인:

```bash
lsof -ti:8000
```

아무것도 출력되지 않으면 포트가 해제된 것입니다.

## 서버 재실행

포트가 해제된 후:

```bash
python run.py
```

또는:

```bash
./start.sh
```
