# FridgeAI 프론트엔드 실행 가이드

## 빠른 시작

### 1. Node.js 설치 확인

Node.js 18 이상이 설치되어 있어야 합니다.

```bash
node --version
```

### 2. 의존성 설치

```bash
cd frontend
npm install
```

### 3. 환경 변수 설정 (선택사항)

`.env.local` 파일을 생성하여 백엔드 API URL을 설정할 수 있습니다:

```bash
cp .env.local.example .env.local
```

`.env.local` 파일 내용:
```
NEXT_PUBLIC_API_URL=http://localhost:8000
NEXT_PUBLIC_SPLINE_SCENE_URL=your-spline-scene-url
```

### 4. 개발 서버 실행

```bash
npm run dev
```

프론트엔드가 `http://localhost:3000`에서 실행됩니다.

### 5. 백엔드 서버 실행

다른 터미널에서 백엔드 서버를 실행하세요:

```bash
# 프로젝트 루트에서
python run.py
```

백엔드가 `http://localhost:8000`에서 실행됩니다.

## 문제 해결

### 포트 충돌

포트 3000이 이미 사용 중인 경우:
```bash
# 포트 3000 사용 중인 프로세스 확인
lsof -ti:3000

# 프로세스 종료
kill -9 $(lsof -ti:3000)
```

또는 다른 포트 사용:
```bash
npm run dev -- -p 3001
```

### 의존성 설치 오류

```bash
# node_modules 삭제 후 재설치
rm -rf node_modules package-lock.json
npm install
```

### Spline 씬 로드 오류

Spline 씬이 로드되지 않는 경우:
1. 인터넷 연결 확인
2. Spline 씬 URL 확인
3. 브라우저 콘솔에서 에러 확인
4. 대체 배경이 자동으로 표시됩니다

### API 연결 오류

백엔드 서버가 실행 중인지 확인:
```bash
curl http://localhost:8000/health
```

CORS 오류가 발생하는 경우, `src/api/main.py`의 CORS 설정을 확인하세요.

## 프로덕션 빌드

```bash
npm run build
npm start
```

빌드된 파일은 `.next` 디렉토리에 생성됩니다.
