# 네트워크 문제 해결 가이드

## 현재 상황
인터넷 연결이 없어 패키지 설치가 실패했습니다.

## 해결 방법

### 1. 인터넷 연결 확인
- Wi-Fi 또는 이더넷 연결 확인
- VPN이 켜져 있다면 일시적으로 해제
- 방화벽 설정 확인

### 2. 네트워크 연결 후 실행

터미널에서 다음 명령어를 순서대로 실행하세요:

```bash
cd /Users/kimhyeseon/Dev/FridgeAI

# 가상환경 활성화
source venv/bin/activate

# 의존성 설치
pip install -r requirements.txt
```

### 3. 설치 확인

```bash
# 가상환경 활성화 상태에서
pip list | grep -E "uvicorn|fastapi|langgraph"
```

다음 패키지들이 보여야 합니다:
- uvicorn
- fastapi
- langgraph
- langchain
- openai
- chromadb

### 4. 서버 실행

의존성 설치가 완료되면:

```bash
python run.py
```

## 수동 설치 (네트워크 연결 후)

필수 패키지만 먼저 설치하려면:

```bash
source venv/bin/activate

# 필수 패키지 설치
pip install uvicorn fastapi python-multipart python-dotenv openai langgraph langchain langchain-openai chromadb Pillow
```

## 문제가 계속되면

1. **DNS 설정 확인**
   ```bash
   # DNS 서버 확인
   scutil --dns | grep nameserver
   ```

2. **프록시 설정 확인**
   - 시스템 설정 > 네트워크 > 고급 > 프록시

3. **방화벽 확인**
   - 시스템 설정 > 보안 및 개인 정보 보호 > 방화벽

4. **pip 캐시 삭제 후 재시도**
   ```bash
   pip cache purge
   pip install -r requirements.txt
   ```
